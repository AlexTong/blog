mac

Mac 重装原因

由于mac从购买至今未做过重装，垃圾文件堆积，本来就不大的SDD128GB 已所剩无几啦。
还有一个让我下定决心重装的是原因是，安装node canvas 安装不成功。xcode升级7.0也无法升级。

前期准备

很重要，就是：备份。
自从买了mac本人一直坚持Time Machine备份，但是为了数据0丢失，重装前再一次执行Time Machine
必备的重装环境那就是：20MB以上的宽带。Mac系统的重装都是需要实时在线认证和下载最新系统镜像文件的。

开始安装文件

第一步先格式化磁盘要不然，Mac直接安装是增量的。所以需要先格式化。不过数据会丢失哦。
第二步输入APPLE ID认证
第三步漫长的等………..待！
第四步骤设置电脑用户。

安装必要的软件

作为程序员首先请安装XCode最新版本和xcode command line tools
为什么？ 这里不解释。
我的安装软件列表
XCode
iTerm2(命令工具)
oh-my-zsh
HomeBrew(OS X 不可或缺的套件管理器)
Dash (程序员好帮手)
Chrome浏览器
Sublime Text(最性感的代码编辑器)
WebStorm(很好用的前段编码神器)
NodeJS(brew install node)
Office 2016(本人购买了家庭版，这也是我电脑里唯一的正版软件，不要告诉别人哦！)
搜狗输入法(目前最好的输入法，几年前我开始使用Mac系统的时候那时候百度输入法还不错，但现在还是搜狗。为什么？大家比我都懂所以也是不解释)
TheUnarchiver （解压缩软件）
GitHub Desktop 也是挺好用的软件(刚开始没有这软件那时候git都是命令行)
MicrosoftRemoteDesktop 比较好用的远程桌面链接软件，要是需要远程桌面链接可以备一个。
Adobe Photoshop
Adobe illustrator

安装字体

作为程序员难免使用Adobe Photoshop和Adobe illustrator改图和修图的工作。
所以尽量丰富自己的电脑的字体库。
比较常用的字体微软雅黑这个最好是装上，因为国内的网站设计很多的默认字体是微软雅黑。
Adobe Source Code Pro 字体是Adobe专门为程序员设计的字体，个人比较喜欢在IDE中使用此字体。
